# DSA Folder Structure and Routes

Below is the list of all folders created for your DSA preparation, along with their recommended study order (route):

## 1. Arrays
- Path: DSA/arrays
- Start here. Practice basic to advanced array problems.

## 2. String
- Path: DSA/string
- Next, cover string manipulation and pattern matching problems.

## 3. Linked List
- Path: DSA/linked_list
- Study singly, doubly, and circular linked lists.

## 4. Stack
- Path: DSA/stack
- Practice stack operations and applications.

## 5. Queue
- Path: DSA/queue
- Learn about queues, circular queues, and priority queues.

## 6. Hashing
- Path: DSA/hashing
- Master hash tables, sets, and related problems.

## 7. Tree
- Path: DSA/tree
- Study binary trees, BST, AVL, heap, and trie.

## 8. Graph
- Path: DSA/graph
- Practice BFS, DFS, shortest paths, and graph algorithms.

## 9. Recursion
- Path: DSA/recursion
- Understand recursion, base cases, and recursive patterns.

## 10. Backtracking
- Path: DSA/backtracking
- Practice backtracking for permutations, combinations, and puzzles.

## 11. Dynamic Programming
- Path: DSA/dynamic_programming
- Learn memoization, tabulation, and classic DP problems.

## 12. Greedy
- Path: DSA/greedy
- Study greedy algorithms and their applications.

## 13. Searching
- Path: DSA/searching
- Practice linear, binary, and advanced search techniques.

## 14. Sorting
- Path: DSA/sorting
- Learn all major sorting algorithms.

## 15. Patterns
- Path: DSA/patterns
- Master DSA patterns. Each subfolder covers a specific pattern:
    - two_pointers
    - sliding_window
    - fast_and_slow_pointers
    - merge_intervals
    - cyclic_sort
    - in_place_reversal
    - tree_bfs
    - tree_dfs
    - two_heaps
    - subsets
    - modified_binary_search
    - bitwise_xor
    - top_k_elements
    - k_way_merge
    - backtracking
    - dynamic_programming
    - greedy
    - topological_sort
    - trie
    - union_find
    - graph_bfs
    - graph_dfs
    - matrix
    - palindromic_substring
    - intervals
    - counting
    - frequency_map
    - prefix_sum
    - monotonic_stack
    - monotonic_queue
    - disjoint_set
    - segment_tree
    - fenwick_tree
    - suffix_array
